## Skill `prd` — start fazy 1

Karta i opis: [skills.sh/github/awesome-copilot/prd](https://skills.sh/github/awesome-copilot/prd)

Instalacja w projekcie:

```bash
npx skills add https://github.com/github/awesome-copilot --skill prd
```

W narzędziu z podpiętymi skills (np. Copilot, inny agent) wołasz procedurę **prd** — formuła zależy od klienta („użyj skilla prd”, „discovery według prd”).

---

## Przykład: pierwsza wiadomość

```text
Używam skilla prd — zacznij Fazę 1 (discovery). Poniżej mój ludzki szkic z trzech bloków (problem / MVP / aktorzy). Zadaj pierwsze pytanie zgodnie z workflow skilla.

---
[wklej tu treść z 1a albo z modala „onTatami”]
---
```

Dalej odpowiadasz na **krótkie tury pytań**; skill pilnuje, żeby nie pisać PRD, dopóki wywiad nie domknie tematu.
