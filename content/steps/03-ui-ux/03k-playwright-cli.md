# Playwright CLI

Narzędzie do automatycznej walidacji działania frontendu.

- **Link (GitHub):** https://github.com/microsoft/playwright-cli
- **Co daje:** testowanie flow użytkownika i interakcji (formularze, nawigacja, edge-case'y).
- **Rola w UI/UX:** nie tylko „ładnie wygląda”, ale też działa stabilnie w realnych scenariuszach.
- **Kiedy używać:** po każdej większej zmianie frontendowej i przed demo/release.
