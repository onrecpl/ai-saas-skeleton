# Krok 3: UI/UX workflow dla Claude Code

Po decyzjach o stacku i infrastrukturze warto od razu dołożyć osobny krok o **jakości frontendu**. To moment, w którym:

- unikasz „AI slop” (generyczne layouty i style),
- dobierasz narzędzia do estetyki i typu produktu,
- zabezpieczasz jakość interakcji oraz użyteczność.

Źródło: analiza filmu **Top 10 Claude Code Frontend Design Skills, Plugins, & CLIs** (Chase AI, 20:07).
