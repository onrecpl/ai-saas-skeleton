# SkillUI

Narzędzie do zamiany stylu istniejącej strony na skill dla Claude Code.

- **Link (GitHub):** https://github.com/amaancoderx/npxskillui
- **Co daje:** analizuje layout i komponenty wybranego serwisu, a potem tworzy „przepis” na podobny styl.
- **Tryb rozszerzony:** może używać Playwright do przechwytywania interakcji i zachowań, nie tylko statycznego HTML.
- **Kiedy używać:** gdy chcesz szybko wystartować z estetyką inspirowaną konkretnym produktem.
